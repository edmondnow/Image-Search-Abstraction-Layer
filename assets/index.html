FreeCodeCamp API: Image Abstraction Layer

User Stories:


  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.

  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.

  • I can get a list of the most recently submitted search strings.

Example input

https://searchforthis.glitch.me/search?text=cow&per_page=1&page=2

Example output

[{"url":"https://farm5.staticflickr.com/4698/26097163758_c714053896.jpg","title":"Mercer Vu Farms in Franklin County, Pa."}]

Example history input

https://searchforthis.glitch.me/history?number=2

Example history output
